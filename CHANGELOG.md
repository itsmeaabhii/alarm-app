# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-06

### Added
- Initial release of Alarm App
- Create and manage multiple alarms
- Enable/disable alarms with toggle switches
- Edit and delete existing alarms
- Persistent storage using SharedPreferences
- Local notifications support
- Custom alarm sounds
- Dark theme with Material Design 3
- Beautiful UI with Google Fonts (DM Sans)

### Features
- Time zone support for accurate alarm scheduling
- Permission handling for notifications and alarms
- Audio playback for alarm sounds
- Clean and intuitive user interface
