# Alarm App

A simple and elegant Flutter alarm application.

## Features

- Create multiple alarms
- Set custom alarm times
- Enable/disable alarms with a toggle
- Edit existing alarms
- Delete alarms
- Persistent storage of alarms
- Clean and intuitive UI

## Getting Started

1. Install dependencies:
   ```
   flutter pub get
   ```

2. Run the app:
   ```
   flutter run
   ```

## Requirements

- Flutter SDK >=3.0.0
- Dart SDK >=3.0.0
