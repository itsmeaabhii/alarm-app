# Alarm App 🔔

A beautiful and elegant Flutter alarm application with a modern dark theme and intuitive user interface.

## ✨ Features

- ⏰ Create and manage multiple alarms
- 🎯 Set custom alarm times with ease
- 🔄 Enable/disable alarms with a simple toggle
- ✏️ Edit existing alarms
- 🗑️ Delete alarms you no longer need
- 💾 Persistent storage of alarms (survives app restarts)
- 🎨 Clean and intuitive Material Design 3 UI
- 🌙 Beautiful dark theme
- 🔔 Local notifications support
- 🔊 Custom alarm sounds

## 🚀 Getting Started

### Prerequisites

- Flutter SDK >=3.0.0
- Dart SDK >=3.0.0
- iOS 12.0+ or Android 6.0+

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd alram
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## 🛠️ Tech Stack

- **Flutter** - Cross-platform framework
- **Material Design 3** - Modern UI design system
- **Google Fonts** - Beautiful typography (DM Sans)
- **SharedPreferences** - Local data persistence
- **Flutter Local Notifications** - Notification system
- **AudioPlayers** - Alarm sound playback
- **Timezone** - Time zone support
- **Permission Handler** - Runtime permissions
